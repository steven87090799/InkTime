# InkTime 內建繁體中文字型

| `LXGWWenKaiTC-Regular.ttf` | 文青風格 | LXGW WenKai TC v1.522 | [lxgw/LxgwWenKaiTC](https://github.com/lxgw/LxgwWenKaiTC/tree/v1.522) | SIL Open Font License 1.1；見 `LXGWWenKaiTC-OFL.txt` |
