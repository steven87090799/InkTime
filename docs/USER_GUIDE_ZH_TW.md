# 使用者指南

## 儀表板與照片

儀表板顯示照片、分析失敗、重複、工作、Token、成本與最近錯誤。照片庫每頁最多 60 張，可搜尋路徑／描述，並依狀態、類型、回憶分與重複群組篩選；每張卡片同時顯示模型描述與電子紙短句。詳細頁會逐項列出本機預篩選實際數值、目前門檻、是否命中與最終決定，並顯示每次模型分析的分類、四項分數、理由、文字結果、電子紙短句、原始 JSON、Token、成本與錯誤。

## 分析工作

administrator 可建立本地、低成本、高品質或兩階段工作。啟動前確認預估照片數與預算。`pausing` 表示正在完成已送出的少量工作；進入 `paused` 後不再 claim。取消後不再送新請求；失敗項目可重跑。viewer 只能查看與匯出。

## 渲染與裝置

「渲染」內建芫荽手寫風格與霞鶩文楷 TC 文青風格，可先看實際字型預覽再切換；管理員也能上傳其他 TTF／OTF／TTC。正式發布仍會逐段檢查所有短文案與地點字元，缺字即明確失敗，不會改用不支援繁中的預設字型。照片含 GPS 時預設顯示最近城市，不會顯示精確座標，也能從「設定」停用。發布會建立不可變版本；回滾只切換 latest。新增裝置後 Token 只顯示一次；遺失時重新產生，舊 Token 立即失效。

## 錯誤與診斷

先記下穩定錯誤碼，再看錯誤中心的聚合次數、工作與照片。診斷頁可查看資源、資料庫、WAL、快取、Worker 與 Queue；診斷包已排除 Secret、私人路徑與 GPS。
