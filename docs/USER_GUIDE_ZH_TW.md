# 使用者指南

## 儀表板與照片

儀表板顯示照片、分析失敗、重複、工作、Token、成本與最近錯誤。照片庫每頁最多 60 張，可搜尋路徑／描述，並依狀態、類型、回憶分與重複群組篩選；每張卡片同時顯示模型描述與電子紙短句。詳細頁會逐項列出本機預篩選實際數值、目前門檻、是否命中與最終決定，並顯示每次模型分析的分類、四項分數、理由、文字結果、電子紙短句、原始 JSON、Token、成本與錯誤。

## 分析工作

administrator 可建立本地、低成本、高品質或兩階段工作。啟動前確認預估照片數與預算。`pausing` 表示正在完成已送出的少量工作；進入 `paused` 後不再 claim。取消後不再送新請求；失敗項目可重跑。viewer 只能查看與匯出。

## 渲染與裝置

「渲染」的智慧裁切工作室會顯示正式 E6 六色量化預覽、裁切焦點，以及對比、主體、膚色、文字／細節四項分數。系統先保護人臉或主體；管理員可調整水平／垂直位置後儲存，也可恢復自動裁切。可預覽並設為預設的版型包括全版照片、明信片、照片＋日期地點、月曆、天氣＋室內溫溼度，所有調整都不修改原始照片。

自動選片預設為「歷年今日」：先選過去年度同月同日的照片，不足時才取前後 7 日及綜合排名候選；範圍與回退規則可在「設定」修改。E6 適合度在模型前以本機演算法計算，過低時可直接排除而不消耗 Token；最愛照片不受此排除。照片含 GPS 時，支援文字的版型可顯示最近城市，不會顯示精確座標。

「渲染」另內建芫荽手寫風格與霞鶩文楷 TC 文青風格，可先看實際字型預覽再切換；管理員也能上傳其他 TTF／OTF／TTC。正式發布仍會逐段檢查短文案、日期、地點與天氣字元，缺字即明確失敗，不會改用不支援繁中的預設字型。發布會建立不可變版本；回滾只切換 latest。新增裝置後 Token 只顯示一次；遺失時重新產生，舊 Token 立即失效。
